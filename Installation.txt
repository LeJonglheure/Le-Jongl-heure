CD   " Le Jongl'heure ".

1. Conseils pour l'utilisation du CD «Le Jongl'heure».

Pour que le programme puisse s'exécuter, il est nécessaire de disposer sur son ordinateur d'une version récente de JAVA (Sun). 

Vous pouvez notamment la télécharger à l'adresse suivante :
http://www.java.com/fr/download/index.jsp
Choisissez la version Java selon votre système d'exploitation, téléchargez-la et installez-la.

Après avoir inséré le CD, choisissez l'option «Ouvrir le dossier pour afficher les fichiers».
Ensuite, cliquez sur le fichier choisi :
soit " Installation " pour connaître les Conseils et les Droits d'utilisation du CD " Le Jongl'heure", soit " Manuel " pour découvrir le Règlement du Jeu " Le Jongl'heure " sur CD ; soit  " LeJonglheurepourDebutants ", soit " LeJonglheure " pour commencer une partie du jeu selon l'expérience des joueurs.
